# Fract
First try to create a web toolbox using fractal

After configuring fractal by fractal.js

Start fractal with:
```bash
$ fractal start --sync
```
## Using handlebars
Use handlebars for extra rendering helpers. 

First install handlebars by:
```bash
$ npm install --save @frctl/handlebars
```
